Terminal:
			
	pwd - выводит текущую дерикторию
	mkdir - создает новую дерикторию
	cd - изменить текущий каталог
	history - вывести историю вводимых команд за все время
	ls - просмотр содержимого В дериктории
	touch - используется для создания файлов
	> - используется для создания файла
	vim - простой текстовый редактор
	ls-la - пишется через пробел опция la выводит все файлы списком явные и скрытые
	mv - испольщуется для перемещения файлов из каталогов
	cp - используется для копирования файлов
	find - используется для поска файлов и других свойств
	grep - находит на вводе строки овечающему заданному регулярному выражению
	tail - выводящая несколько по умолчанию 10 последних строк из файла
	head - выводящая первые n строк из файла по умолчанию n равно 10
	date - выводит_текущую_дату
	curl - позволяет подключиться к серверу по множеству разных протоколов
	cat - создает файлы позволяет редактировать файл без использования vim
	explore - открывает текущую дерикторию
		
gitbush:

	git clone - кланирует репозиторий
	git add - первая команда в цепочке сохранения текущего состояни проект
	git commit -m "название комита" - команда для записи индексированных изменений в репозиторий
	git push - отправить изменения в репозиторий
	git pull -получение файлов из удаленного репозитория
	git brench - используется для просмотра веток и их создания
	git checkout - используется для перехода по веткам
	git merge - используется для слияние веток 
	git fetch -  загружает коммиты, файлы и ссылки из удаленного репозитория в ваш локальный репозитори
